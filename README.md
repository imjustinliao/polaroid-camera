# Polaroid Camera

A web-based Polaroid camera simulator that captures and displays photos with authentic retro styling.

## Features

- Interactive camera interface with realistic Polaroid aesthetics
- Photo capture and display
- Responsive design for mobile and desktop devices
- Vintage-inspired styling and animations

## Getting Started

Simply open `index.html` in your web browser to start using the Polaroid camera simulator.

## Development

This is a standalone web application built with HTML, CSS, and JavaScript.
